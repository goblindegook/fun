# reduceRight

Simple modules for functional programming, done for fun.

## Usage

```javascript
import reduceRight from 'funny-reduce-right'

reduceRight((acc, item) => acc + item, [1, 2, 3, 4])     // 10
reduceRight((acc, item) => acc + item, [1, 2, 3, 4], 10) // 20
```

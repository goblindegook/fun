# reduce

Simple modules for functional programming, done for fun.

## Usage

```javascript
import reduce from 'funny-reduce'

reduce((acc, item) => acc + item, [1, 2, 3, 4])     // 10
reduce((acc, item) => acc + item, [1, 2, 3, 4], 10) // 20
```
